Etiquetas extra

Etiqueta pre

la etiqueta p es para respetar espacios y lineas,al contrario del parrafo

hola soy roberta tengo 16 años 

 hola               soy           roberta 
        tengo        16      años

Etiqueta abr

la etiqueta de abr se utiliza para abrevievatura

segun los datos de la OMS El consumo excesivo de azúcar provoca daños en la salud

Etiqueta bdo

si se desea cambiar la direccion de un texto se utiliza la etiqueta bdo

Esto es un texto normal

este es un texto diferente

Etiqueta cite

la etiqueta cite se utiliza para escribir citas textuales

lo unico que se es que no se nada por: pinterest

etiqueta small

hace la letra mas pequeña

este texto esta dentro de una etiqueta small

Etiqueta sub

si desea escribir un subidice se tiene que usar la etiqueta sub

983 8

Etiqueta sup

si se decea escribir un supindice tiene que usar la etiqueta sup

Etiqueta TIME

para escribir fechas y horas se debe usar la etiqueta time

Mañana es 6 de noviembre del 2024
